# turbo-octo-barnacle
grinder game test bed

## To play!!!

Run the following commands from repo root as needed:

1. Initialize the project after clone (only needed once)
    ```
    $ bin/init
    ```

2. Launch local instance of the game
    ```
    $ bin/playlocal
    ```

2. Launch web instance of the game
    ```
    $ bin/playweb
    ```
